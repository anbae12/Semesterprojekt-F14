Koordinattransformationen fra kartesiske til sværiske kan gøres ud for nedenstående ligninger.


\begin{align}
\begin{split}
\rho &=\sqrt { { x }^{ 2 } +{ y }^{ 2 }+{ z }^{ 2 }} 
\\
\phi &={ tan }^{ -1 }\left( \frac { \sqrt { { x }^{ 2 }+{ y }^{ 2 } }  }{ z }  \right) 
\\
\theta &={ tan }^{ -1 }\left( \frac { y }{ x }  \right) 
\label{eq:sv_koordi}
\end{split}
\end{align}
